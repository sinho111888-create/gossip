# 易经运程 (I Ching Daily index)

基于《周易》六十四卦的每日运势占卜网页应用。输入姓名，即可获得基于姓名与当日日期的专属卦象与运势解读。

## 如何使用

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖。

或者通过本地静态服务运行：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000/index.html
```

## 功能

- **每日卦象** — 16 卦精选，包含卦名、上下卦、爻象解读
- **今日总运** — 基于卦辞的整体运势
- **四大运势** — 事业/学业、财运、感情/人际、健康，各有吉凶等级 (good/mid/warn/bad)
- **易经名句** — 随机引用《周易》经典句子
- **确定性算法** — 同一人同一日结果相同，不同人/不同日结果不同

## 原理

通过 `hash(姓名) XOR hash(日期)` 生成的组合哈希值，从卦象、运势、名言等数据集中选取对应条目，确保每日每人的占卜结果**确定且有区分度**。

## 技术栈

- 纯 HTML + CSS + JavaScript，零外部 JS 依赖
- 字体：马善政体 (标题) + Noto Serif SC (正文)，通过 Google Fonts 加载
- 深色古典中国风视觉设计

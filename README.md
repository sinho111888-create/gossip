# 每日一占

心中默想两个数字，以四法同占，一窥今日天机。

## 占卜方法

- **小六壬** — 以两数顺逆落宫，得六宫之一，断吉凶方位
- **梅花易数** — 两数定上下卦与动爻，得本卦·互卦·变卦，体用生克
- **六爻占卜** — 两数生成六爻动静，得本卦变卦，观爻象之变
- **大六壬** — 两数择十二天将，得天将神谕

## 如何使用

直接用浏览器打开 `index.html`。

或本地服务：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```

- 纯 HTML + CSS + JavaScript，零外部依赖
